MineMods Manager
Sistema para gerenciamento de mods de Minecraft desenvolvido em React.

Funcionalidades
Cadastro de mods
Listagem de mods
Busca por nome
Edição de mods
Exclusão de mods
Armazenamento com LocalStorage
Consumo da API Modrinth
Navegação com React Router
Tecnologias
React
JavaScript
CSS
React Router DOM
Fetch API
LocalStorage
Como executar
npm install

npm run dev
